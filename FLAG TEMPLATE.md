# **Flag Template — Agent Audit Checklist**

*Only flag if it matches one of the cases below. Nothing else.*

This checklist defines exactly when to flag an agent conversation. Read each flag carefully, including the examples. When in doubt, refer to the "Do Not Flag" example first.

## **Flag 1 — Continued After Opt-Out**

**When to flag:** Lead says "stop texting," "remove me," "don't contact me," or "leave me alone" — and the agent sends another message after.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "Stop texting me" Agent: "Want to reconsider selling?" **Flag** | Lead: "No thanks" Agent: "Is it a never or maybe later?"OR any phrasing with the same meaning **No flag — soft rejection, not an opt-out** |

## **Flag 2 — Aggressive or Deceptive Language**

**When to flag:** Agent uses threats, profanity, intimidation, or false claims. Examples: "I'll report you," using profanity, or claiming a fake buyer is ready.

| Flag This | Do Not Flag |
| :---- | :---- |
| Agent: "If you don't respond, I'll file a report" **Flag** | Agent: "We buy houses for cash" Agent: "We can close quickly"OR any phrasing with the same meaning **No flag — normal business language** |

## **Flag 3 — Stated a Firm Dollar Offer**

**When to flag:** Agent gives a specific price, not a range. Example: "$185,000" or "I can do $200k."

| Flag This | Do Not Flag |
| :---- | :---- |
| Agent: "We can offer you $185,000" **Flag** | Agent: "Between $140k–$175k depending on condition" Agent: "Let me work on a number for you" Agent: "I pay $1,000 for referrals" OR any phrasing with the same meaning **No flag — range, working on it, or referral close** |

## **Flag 4 — Gave Up After First "No"**

**When to flag:** Lead says "not interested," "no," "no thanks," or "not for sale" — and the agent stops or just says "ok bye" without a rebuttal.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "Not interested" Agent: "Ok, have a good day\!" **Flag** | Lead: "Not interested" Agent: "No worries. Is it more a never, or maybe later?" OR any phrasing with the same meaning **No flag — agent used a rebuttal** |

## **Flag 5 — Wrong Number But Kept Selling**

**When to flag:** Lead says "wrong number," "I don't own that," or "not my property" — and the agent continues asking about the property.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "Wrong number, I don't own it" Agent: "What price would you need?" **Flag** | Lead: "Wrong number" Agent: "Sorry\! Know anyone who wants to sell? I pay $1k referrals" OR any phrasing with the same meaning **No flag — agent switched scripts** |

## **Flag 6 — Scheduled Call With No Info**

**When to flag:** Lead asks to call — and the agent says "yes, when?" without asking any qualifying questions first.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "Can you just call me?" Agent: "Sure\! When's good?" **Flag** | Lead: "Can you just call me?" Agent: "Before I call, what's the condition of the house?"OR any phrasing with the same meaning **No flag — agent pre-qualified first** |

## **Flag 7 — Revealed or Promised Over 6 Months**

**When to flag:** Agent states the timeline themselves, e.g. "we work within 6 months" or "we can wait 8 months."

| Flag This | Do Not Flag |
| :---- | :---- |
| Agent: "We typically close within 6 months" **Flag** | Agent: "We're flexible on timing" Lead: "I need 6 months" — Agent: "Perfect\!"OR any phrasing with the same meaning **No flag — vague language or lead set the timeline** |

## **Flag 8 — Incoherent Message or Wrong Name**

**When to flag:** Agent sends a message that is unreadable, uses the wrong person's name, or makes no sense in context.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead's name: Robert Agent: "Hey Karen\! So about the Elm St property, should u sell or not..." **Flag — wrong name used** | Agent: "u still thinking about selling?" **No flag — casual abbreviations are fine** |

## **Flag 9 — Ignored Clear Interest**

**When to flag:** Lead shows interest ("yes," "maybe," "tell me more," "how does it work?") — and the agent ignores it and ends the chat.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "I might sell. How does it work?" Agent: "Ok great, have a good day\!" **Flag** | Lead: "How much will you pay?" Agent: "Great\! Before I say, tell me about the property..." **No flag — agent engaged and kept going** |

## **Flag 10 — Tried to Close With Zero Info**

**When to flag:** Lead shows interest — and the agent immediately tries to book a call or make an offer without collecting any information first.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "Maybe I'd consider it" Agent: "Perfect\! When can we hop on a call?" (no property details collected) **Flag** | Lead: "I could sell. House needs a roof, want $280k, need 2 months" Agent: "Great\! When's a good time to call?" **No flag — lead gave info first** |

## **Flag 11 — Did Not Escalate After Full Info**

**When to flag:** Lead gives all 4 pieces of information (condition, price, motivation, timeline) — and the agent does not try to book a call or move forward.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "3 bed 2 bath, needs work, want $220k, divorce, need out by summer" Agent: "Got it. Thanks for sharing. Let me know if you have questions\!" **Flag — all 4 pieces given, no escalation** | Lead: "House needs work, $220k" Agent: "Got it\! What about motivation and timeline?" **No flag — only 2 pieces, agent still gathering** |

##           **Flag 12 — No Referral Close After High Price**

**When to flag:** Lead asks for an unrealistic or high price — and the agent does not offer the $1,000 referral before ending the conversation.

| Flag This | Do Not Flag |
| :---- | :---- |
| Lead: "I want $900k for my 2-bed condo" Agent: "That's above what we can do. Bye\!" **Flag** | Lead: "I want $900k" Agent: "That's above market. But know anyone else? I pay $1k referrals" **No flag — agent included the referral close** |
